Вход на сайт hotels.com

Тест 1: +
При вводе случайного набора символов в поле города, сайт выдает ошибку о несуществовании такой локации
1) Город - случайный набор символов (например "фвфцвфцвфцафцвф"), 
   даты заезда,отъезда - любые не прошедшие (например 25/12/2019 - 30/12/2019),
   один номер на 2 взрослых
2) нажимаем "Поиск"
Итог: Появилось сообщение "К сожалению, нам не удалось найти фвфцвфцвфцафцвф"

Тест 2: +
При вводе прошедшей даты не будет найдено свободных номеров
1) Выберем Минск, с 07/12/2019 по 10/12/2019 один номер на 2 взрослых
2) Нажимаем "поиск"
Итог: во всех отелях нет свободных мест на эту дату

Тест 3: +
Дожен быть хотябы один номер в отеле 5 звезд с бесплатным wifi
1) Выберем Минск, с 10/12/2019 по 20/12/2019 один номер на 2 взрослых, поиск
2) Выберем 5 звезд и бесплатный wifi
Итог: нашло несколько гостиниц с подходящими параметрами

Тест 4: +
Для заказа доступен хотябы один номер на завтрашнюю дату
1) Выберем Минск, с завтрашнего дня, на 1 неделю, один номер на 2 взрослых, поиск
2) Выберем первый доступный отель и первую комнату в нем
3) переходим к заполнению данных и оплате
Итог: Можно заказать номер на завтра

Тест 5: +
При смене валюты, цены должны показываться в ее условных единицах
1)Сменим валюту на доллары
2)Выберем Минск, с 10/12/2019 по 20/12/2019 один номер на 2 взрослых
3)Выполним поиск
Итог: цены номеров показываются в долларах

Тест 6: *Завезли капчу, драйвером не войти в аккаунт*
Мой список бронирований должен быть пуст
1) Входим в подготовленный аккаунт
2) Выбираем "Мои бронирования"
Итог: Страница сообщает что нет бронирований

Тест 7: +
При выборе спецпредложений в результатах поиска должно быть хотябы одно спецпредложение
1) Выбираем "спецпредложения"
2) Выберем Лондон с 20/12/2019 по 25/12/2019 один номер на 2 взрослых
3) нажимаем поиск
Итог: Все доступные отели идут с меткой спец предложения

Тест 8: +
Должна быть хотябы одна квартира для бронирования
1) Выберем Минск с 20/12/2019 по 25/12/2019 один номер на 2 взрослых
2) Нажимаем поиск
3) выбираем тип отеля - квартира, ждем завершения поиска
Итог: есть несколько доступных квартир

Тест 9: +
При вводе названия существующего города сайт покажет результаты для него
2) Выберем Лондон с 20/12/2019 по 25/12/2019 один номер на 2 взрослых
3) нажимаем поиск
Итог: Сайт нашел множество отелей в этом городе
